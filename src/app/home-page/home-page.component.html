ProSeqViewer

A TypeScript library to visualize annotation on single sequences and multiple sequence alignments
ProSeqViewer can be integrated in both modern and dynamic frameworks like Angular as well as in static HTML websites. It is used by MobiDB, DisProt and RepeatsDB
