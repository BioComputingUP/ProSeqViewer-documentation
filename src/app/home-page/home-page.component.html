    ProSeqViewer

    Library for the visualization of (not only) biological sequence features.

    Used in MobiDB, DisProt and RepeatsDB.

    Based on Typescript and compatible with Angular 2+ framework. A JavaScript version of the package is also available (see the Documentation).
