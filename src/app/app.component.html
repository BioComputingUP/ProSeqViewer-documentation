Sequence Viewer Typescript

This is the full documentation of the Sequence Viewer Typescript library for the visualization of (not only) biological sequence features: sequence-viewer-typescript (Github). Used in MobiDB, DisProt and RepeatsDB.

This package is based on Typescript and compatible with Angular 2+ framework. A Javascript version of the package is also available (see the API documentation for more info).

Lateral Indexes example

Set a lateral starting index for the sequence/s. You can count gaps or not. (See API Documentation)

Here the start index is set at 2 and gaps are counted.

The sequence-viewer width is set to 450px to show the indexes behaviour when the sequence is split in several lines.


         <head>

         </head>
        <div id ="sqvLateralIndexes"></div>
         <script src="sqv-bundle.js"</div>
        <script>
          const sequenceviewer = new SequenceViewer('sqvLateralIndexes'); 
          const sequences = [ 
                {sequence: 'MVLSPADKTNVKAA-----EYGAEALERMFLSFPTTKTYFPHFDLSHGSAQVKGHG', id: 1, label: 'sp|P69905|HBA_HUMAN', startIndex: 2}
                ];
          const options =  {
                lateralIndexesGap: true
                };
          sequenceviewer.draw(sequences, options);
        </script>

Selection example

1. Copy-paste event

Click and drag the mouse to select a region, the region will be highlighted in yellow. Use the shortcut Ctrl+C to copy and Ctrl+V to paste. The selected region will disappear after using a shortcut or clicking on another area.

1. On character selected event

The double click event will return information about the clicked character, like the x, y positions and the character itself.


     window.addEventListener('onRegionSelected', (e) =>   {
             console.log('character selected', e);
           });

Installation Guide - Javascript

1. Download the sequence viewer stylesheet

You can find the sqv.css file in the src/assets folder on the Github repo.

2. Download the library code

You can find the sqv-bundle.js file in the src/assets folder on the Github repo.

3. Load the stylesheet

Place the following code in the <head></head> section.

<link rel="stylesheet" type="text/css" href="sqv.css"></div>

4. Load the library

<script src="sqv-bundle.js"</div>

5. Place the sequence viewer in your index.html

Be sure to place the following code before the <script></script> section.

<div id ="sqv"></div>

6. Create an instance of the sequence viewer in javascript

Place the following code in the <script></script> section.

const sequenceviewer = new SequenceViewer('sqv');

7. Add inputs

Here an example with minimum required input, for the full list of input settings options check the available parameters

const sequence = [{sequence: 'DFRLE--F-DFRLEDFRLE----}]

8. Execute the function to draw the sequence viewer on the page 

sequenceviewer.draw(sequence);

Installation Guide - Angular

The sequence-viewer-typescript package is written in pure typescript and does not have any dependencies. This guide focus on how to install the package in an Angular app.

1. Install the library using npm

npm i sequence-viewer-typescript

2. Load the sequence viewer stylesheet in your angular.json "styles" 

styles: ["./node_modules/sequence-viewer-typescript/dist/assets/sqv.scss"]

3. Import the sequence viewer in your angular component 

import { SequenceViewer } from 'sequence-viewer-typescript/dist';

4. Place the sequence viewer in your html 

<div id ="sqv"></div>

5. Create an instance of the sequence viewer in your app.component.ts

const sequenceviewer = new SequenceViewer('sqv');

6. Add inputs

Here an example with minimum required input, for the full list of input settings options check the available parameters.

const sequence = [{sequence: 'DFRLE--F-DFRLEDFRLE----}]

7. Execute the function to draw the sequence viewer on the page 

sequenceviewer.draw(sequence);
