ProSeqViewer

ProSeqViewer library for the visualization of (not only) biological sequence features. Used in MobiDB, DisProt and RepeatsDB.
Based on Typescript and compatible with Angular 2+ framework. A JavaScript version of the package is also available (see the API documentation).

Selection example

1. Copy-paste event

Click and drag the mouse to select a region, the region will be highlighted in yellow. Use the shortcut Ctrl+C to copy and Ctrl+V to paste. The selected region will disappear after using a shortcut or clicking on another area.

1. On character selected event

The double click event will return information about the clicked character, like the x, y positions and the character itself.


     window.addEventListener('onRegionSelected', (e) =>   {
             console.log('character selected', e);
           });

API Documentation

Installation Guide - JavaScript

1. Download the ProSeqViewer stylesheet

You can find the sqv.css file in the src/assets folder on the Github repo.

2. Download the library code

You can find the sqv-bundle.js file in the dist folder on the Github repo.

3. In your index.html:

  1. <head>
  2. <link rel="stylesheet" type="text/css" href="sqv.css"> // Load the stylesheet
  3. </head>
  4. <body>
  5. <div id ="sqvSimple"></div>
  6. <script src="sqv-bundle.js"></script>
  7. <script>
  8. const proseqviewer = new ProSeqViewer('sqvSimple'); // Create an instance of the ProSeqViewer in JavaScript
  9. const sequence = [
  10. {sequence: 'MVLSPADKTNVKAAWGKVGAHAG--ALERMFL--YFPHFDLSHGSAQVKGHG', id: 1, label: 'sp|P69905|HBA_HUMAN'}
  11. ];
  12. proseqviewer.draw(sequence); // Execute the function to draw the ProSeqViewer on the page
  13. </script>
  14. </body>
Installation Guide - Angular

The proseqviewer package is written in pure typescript and does not have any dependencies. This guide focus on how to install the package in an Angular app.

1. Install the library using npm

npm i proseqviewer

2. Load the ProSeqViewer stylesheet in your angular.json "styles" 

styles: ["./node_modules/proseqviewer/dist/assets/sqv.scss"]

3. Import the ProSeqViewer in your angular component

  1. import { ProSeqViewer } from 'sequence-viewer-typescript/dist';

4. Place the ProSeqViewer in your template

  1. <div id ="sqv"></div>

5. Create an instance of the ProSeqViewer in your component, add inputs and execute

  1. const proseqviewer = new ProSeqViewer('sqv');
  2. const sequence = [{sequence: 'DFRLE--F-DFRLEDFRLE----}]
  3. proseqviewer.draw(sequence);
