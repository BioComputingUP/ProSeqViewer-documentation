ProSeqViewer

ProSeqViewer library for the visualization of (not only) biological sequence features. Used in MobiDB, DisProt and RepeatsDB.
Based on Typescript and compatible with Angular 2+ framework. A JavaScript version of the package is also available (see the API documentation).

Wrap Line

Each letter is encapsulated in a column element (just one cell for single sequences) which is part of a block of columns. Blocks are graphically separated to improve readability and can have different length (setting chunkSize option). By default, blocks are automatically moved to a new line when they exceed the container width (responsive behavior). However, block wrapping can be disabled if the user wants to slide MSAs horizontally (non-responsive behavior).

Responsive (default)

Non-responsive

Sequence Color Matrix
Set a global sequence color matrix. Available global sequence color matrix is blosum62. Here amino acid pairs are compared, in particular each residue is compared with the residue at the same position in the first sequence. The aminoacid pair is colored based on a input matrix.

Blosum62 is a default matrix, but it is also possible to set a custom matrix.

Blosum62 (default matrix)

  • const options = {
  • sequenceColorMatrix: 'blosum62'
  • };

Custom sequence color matrix

Consensus
A consensus sequence will be the generated on top of the alignment, representing the most frequent aminoacid (consensus identity) or property (consensus physical) found at each position in a sequence alignment.
As a default, the consensusDotThreshold is set at 90%, meaning that position with a frequency less of 90% will display a dot.
In the consensus identity the aminoacid found at each position is colored based on its frequency in the alignment. Each level has a color: (100%, 80%, 70%..)
In the consensus physical the aminoacid found at each position is colored based on the aminoacid property frequency in the alignment. The property symbol is displayed.
Consensus 'identity' and 'physical' can't be set at the same time. In case of overlapping settings, consensus 'identity' will be displayed.

It is also possible to set a custom consensus with personalized colors, levels and symbols and modify the consensusDotThreshold, check the Documentation for details.

Documentation

ProSeqViewer is a TypeScript library implemented to be compatible with modern applications (both Angular and pure JavaScript supported). It is lightweight, fast, interactive, dynamic, responsive and it adapts to any screen size. It generates pure HTML which is compatible with any browser and operating system. The source code is available on Github.

JavaScript installation

1. Download the library code

You can find the sqv-bundle.js file in the dist folder on the Github repo.

2. Download the CSS stylesheet

You can find the sqv.css file in the src/assets folder on the Github repo.

3. Edit your index.html:

  • <head>
  • <link rel="stylesheet" type="text/css" href="sqv.css">
  • </head>
  • <body>
  • <div id ="sqvSimple"></div>
  • <script src="sqv-bundle.js"></script>
  • <script>
  • const proseqviewer = new ProSeqViewer('sqvSimple');
  • const sequence = [
  • {sequence: 'MVLSPADKTNVKAAWGKVGAHAG--ALERMFL--YFPHFDLSHGSAQVKGHG', id: 1, label: 'sp|P69905|HBA_HUMAN'}
  • ];
  • proseqviewer.draw({ sequence});
  • </script>
  • </body>
Angular installation

The proseqviewer package is written in pure typescript and does not have any dependencies. This guide focus on how to install the package in an Angular app.

1. Install the library using npm

npm i proseqviewer

2. Edit the "styles" field in your angular.json 

styles: ["./node_modules/proseqviewer/dist/assets/sqv.css"]

3. Edit your angular component:

  • import { ProSeqViewer } from 'proseqviewer/dist';
  • const proseqviewer = new ProSeqViewer('sqv');
  • const sequence = [{sequence: 'DFRLE--F-DFRLEDFRLE----}]
  • proseqviewer.draw({ sequence});

4. Edit your template:

  • <div id ="sqv"></div>

Events

Three different types of events are available: area selection and column selection (active after the user's Ctrl+C action) and the double click event (active after user's double click).

Area selection

Click and drag the mouse to select a region, the region will be highlighted in yellow. Use the shortcut Ctrl+C to copy and Ctrl+V to paste. The selected region will disappear after using a shortcut or clicking on another area.


     window.addEventListener('onAreaSelected', (e) =>   {
             console.log('event type', e['detail'].type);
             console.log('text area', e['detail'].text);
           });

Columns selection

Double click

The double click event will return information about the clicked character, like the x, y positions and the character itself.


     window.addEventListener('onRegionSelected', (e) =>   {
             console.log('character selected', e['detail'].char);
             console.log('position x', e['detail'].x);
             console.log('position y', e['detail'].y);
           });
