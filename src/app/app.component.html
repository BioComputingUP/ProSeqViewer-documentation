Simple example

Set the sequences and their colorScheme.


        const sequences = [ 
              {sequence: 'MVLSPADKTNVKAAWGKVGAHAGEYGAEALERMFLSFPTTKTYFPHFDLSHGSAQVKGHG', id: 1, label: 'Homo Sapiens'},
              {sequence: 'MVLSGEDKSNIKAAWGKIGGHGAEYGAEALERMFASFPTTKTYFPHFDVSHGSAQVKGHG', id: 2, label: 'Bos Taurus',  colorScheme: 'clustal'},
              {sequence: 'MSLTRTERTIILSLWSKISTQADVIGTETLERLFSCYPQAKTYFPHFDLHSGSAQLRAHG', id: 2, label: 'Mus musculus'}
              ];
        sequenceviewer.draw(sequences);
      

      <div id="sqv"></div>

Regions example

Set the sequences and highlight them with customizable colors.


      const sequences = [ 
            {sequence: 'MVLSPADKTNVKAAWGKVGAHAGEYGAEALERMFLSFPTTKTYFPHFDLSHGSAQVKGHG', id: 1, label: 'Homo Sapiens'},
            {sequence: 'MVLSGEDKSNIKAAWGKIGGHGAEYGAEALERMFASFPTTKTYFPHFDVSHGSAQVKGHG', id: 2, label: 'Bos Taurus'},
            {sequence: 'MSLTRTERTIILSLWSKISTQADVIGTETLERLFSCYPQAKTYFPHFDLHSGSAQLRAHG', id: 2, label: 'Mus musculus'}
            ];
      const regions = [ 
            {sequenceId: 2, start: 1, end: 10, backgroundColor: '#D72638', color: '#FFFFFF'},
            {sequenceId: 2, start: 11, end: 20, backgroundColor: '#FF570A', color: '#FFFFFF'},
            {sequenceId: 2, start: 21, end: 30, backgroundColor: '#FFBC42', color: '#FFFFFF'},
            {sequenceId: 2, start: 31, end: 40, backgroundColor: '#44885B', color: '#FFFFFF'},
            {sequenceId: 2, start: 41, end: 50, backgroundColor: '#393C93', color: '#FFFFFF'},
            {sequenceId: 2, start: 51, end: 60, backgroundColor: '#7E1173', color: '#FFFFFF'}
            ];
      sequenceviewer.draw(sequences, regions);
       

         <div id="sqv"></div>

Patterns example

Match the sequences against a regular expression. 


    const sequences = [ 
          {sequence: 'MVLSPADKTNVKAAWGKVGAHAGEYGAEALERMFLSFPTTKTYFPHFDLSHGSAQVKGHG', id: 1, label: 'Homo Sapiens'},
          {sequence: 'MVLSGEDKSNIKAAWGKIGGHGAEYGAEALERMFASFPTTKTYFPHFDVSHGSAQVKGHG', id: 2, label: 'Bos Taurus'},
          {sequence: 'MSLTRTERTIILSLWSKISTQADVIGTETLERLFSCYPQAKTYFPHFDLHSGSAQLRAHG', id: 2, label: 'Mus musculus'}
          ];
    const patterns = [ 
          {sequenceId: 1, pattern: /MVL/g, backgroundColor: '#D72638'},
          {sequenceId: 1, pattern: /GA/g, backgroundColor: '#D72638'},
          {sequenceId: 2, pattern: /LE/g, backgroundColor: '#FF570A'},
          {sequenceId: 2, pattern: /TT/g, backgroundColor: '#FF570A'},
          {sequenceId: 3, pattern: /AA/g, backgroundColor: '#FFBC42'}
          ];
    sequenceviewer.draw(sequences, patterns);
          

    <div id="sqv"></div>

Lateral indexes example

Set a lateral starting index for the sequence/s. You can count gaps or not. (See API Documentation)

Here the start index is set at 2 and gaps are counted.

The sequence-viewer width is set to 450px to show the indexes behaviour when the sequence is split in several lines.


    const sequences = [ 
          {sequence: 'MVLSPADKTNVKAAWGKVGAHAGEYGAEALERMFLSFPTTKTYFPHFDLSHGSAQVKGHG', id: 1, label: 'Homo Sapiens', startIndex: 2}
          ];
    const options =  {
          lateralIndexesGap: true
          };
    sequenceviewer.draw(sequences, options);
          

    <div id="sqv"></div>

Installation Guide

The sequence-viewer-typescript package is written in pure typescript and does not have any dependencies. This guide focus on how to install the package in an Angular app.

1. Install the library using npm

npm i sequence-viewer-typescript

2. Import the sequence viewer in your angular component 

import { SequenceViewer } from 'sequence-viewer-typescript/dist';

3. Load the sequence viewer stylesheet in your angular.json "styles" 

styles: ["./node_modules/sequence-viewer-typescript/dist/assets/sqv.scss"]

4. Place the sequence viewer in your html 

<div id="sqv"></div>

5. Create an instance of the sequence viewer in javascript and style it

const sequenceviewer = new SequenceViewer('sqv');

6. Add inputs

Here an example with minimum required input, for the full list of input settings options check github documentation

const sequence = [{sequence: 'DFRLE--F-DFRLEDFRLE----}]

7. Execute the function to draw the sequence viewer on the page 

sequenceviewer.draw(sequence);

Available parameters

Field 0 Field 1 Type Example Description
Sequences
sequence string LYRLELERECAMT Sequence of characters
id integer 1 Identifier of the sequence
label string 1xvz_1_30.pdb Lateral label
colorScheme string clustal, gradient, binary, blosum62 Predefined color palette
labelTooltip string <span class="tooltipText" style="background-color: blue;">Tooltip text</span> Stylable tooltip
startIndex integer 30 Start index of the input sequence.
Regions
sequenceId integer 1 Identifier of the sequence.
start integer 10 Start position of the region to color.
end integer 20 End position of the region to color.
backgroundColor string hex, rgb Color the character background.
color string hex, rgb Color the character.
backgroundImage integer linear-gradient(red, yellow) Color the character background with a two color gradient.
Patterns
sequenceId integer 1 Identifier of the sequence.
start integer 10 Start position of the region where to perform the regex.
end integer 20 End position of the region where to perform the regex.
backgroundColor string hex, rgb Color the regex match background.
color string hex, rgb Color the regex match the character.
backgroundImage integer linear-gradient(red, yellow) Color the regex match character background with a two color gradient.
Icons
sequenceId integer 1 Identifier of the sequence.
start integer 10 Start position of the icon region.
end integer 20 End position of the icon region.
icon string lollipop, helix, strand, strand-arrow-left, turn svg image to apply on top of the indicated sequence.
display string center Center the icon in the middle of the start-end region.
Options
fontSize string 14px Font dimension.
chunkSize integer 10 group the sequence into chunks of subsequential characters.
spaceSize integer 2 Space among chunks.
logLevel string debug or none Print console log error with "debug" option.
topIndexes boolean true Place an index on top of the end of each chunk (first row only).
oneLineSetting boolean true Display the viewer with fixed width (not responsive) and scrollbar on the sequences.
oneLineWidth integer 2 Width of sequenceViewer - sidebar.
lateralIndexesGap true Shows lateral indexes including gaps in the count (true) or excluding them (false).
colorScheme integer clustal, gradient, binary, blosum62 Colorscheme to apply to all sequences.
consensusType string identity, physical Add a line with consensus information.
consensusThreshold integer 90 Threeshold for consensus information.
iconsHtml
id string icons Input identifier.
iconName string <svg>...</svg> An svg icon.
